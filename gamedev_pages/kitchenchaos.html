Kitchen Chaos

An endless 3D cooking simulator. To create this game, I followed an online tutorial by
Code Monkey to learn game development in Unity and programming with C#. Turn in as many
dishes as possible within the time limit!
The game uses Unity, C#, and JavaScript to function in browser.

Controls To Play are displayed on the countdown screen before the game starts.
