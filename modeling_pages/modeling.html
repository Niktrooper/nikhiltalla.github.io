About

Here are some CAD and 3d modeling projects I have worked on. These I have worked on to pick up some skills
outside of just computer science which I have found useful in some of my embedded software projects (3D
printing different parts like the chassis).

LEGO Car

A LEGO car modeled first as individual pices then combined into a
single assembly. Basically, built a LEGO car with modeled LEGO pieces.
This taught me alot about contraints and the importance of standard
measuring practices.

X-Wing from Star Wars

This is a 3d model of Poe Dameron's X-Wing from the Star Wars universe.
code to read data from an IMU using UART and export data to a struct
containing magnetometer and other positioning data.
Skills: Python, Beaglebone wiring, UART serial interface

Card Dealing Robot

A small robot capable of dealing cards around it by using a spinning
wheel to eject cards. Required using two motors (rotating and ejecting)
along with buttons to let users set number of players and cards per player.
Skills: C++, Soldering, 3D Modeling, Arduino wiring (with breadboard)
