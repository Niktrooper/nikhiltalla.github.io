Embedded Software

Embedded software development using various languages and skills.
Skills: C++, Soldering, 3D Modeling (Autodesk Inventor) + Printing (Cura)

About

Here are some embedded software development projects I have worked on within the last few years.
These helped me develop skills in programming and resourcefulness (using online resources to understand
electrical components) and explore electrical/embedded engineering through circuit making and soldering.

Arduino Scale

An award winning scale for Science Olympiad measuring mass using an Arduino
(programmed in C++) and a force sensitive resistor. Mass readings are calculated
using a mapping function from voltage differences during calibration tests.
Skills: C++, Circuit Design, Soldering, 3D Modeling + Printing, Data Collection/Analysis

Arduino Salinity Probe

An award winning detector for Science Olympiad measuring solution salinity
using an Arduino and a conductivity probe designed from 3D printed components.
Skills: C++, Circuit Design, Soldering, 3D Modeling + Printing, Data Collection/Analysis

Maze Navigating Robot

An award winning robot for Science Olympiad programmed in C++ to reach target
zones within mazes and reach a target point within the maze while avoiding walls.
Takes readings from gyro sensor (rotation) and utilizes a two-motor drivetrain.
Skills: C++, VEX Robotics, Data Collection/Analysis

Arduino Redox Probe

An award winning detector for Science Olympiad measuring redox potential
of solutions using an Arduino and a redox probe designed from a 3D printed
component, a silver nitrate solution, and a graphite electrode.
Skills: C++, Circuit Design, Soldering, 3D Modeling + Printing, Data Collection/Analysis
