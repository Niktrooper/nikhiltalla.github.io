Blightfall

Blightfall is a tower defense style game where players must use different mechanics including a rover to defend their spaceship from hordes of enemies and attempt to escape the hostile planet they are on. This was developed over a course of two weeks as a rapid prototype concept for the second project in the class EECS 494 at the University of Michigan. Players have to make many decisions, from what towers to build to what to upgrade and even where to keep the rover so that its abilities can be used to prevent failure.

I used Autodesk Inventor to make the models in this game (from the rover to the platforms and attachments), Photoshop for editing pictures, and C# with Unity to develop the game. Althought this was a solo project, I used Git for version control and Atlassian Jira for project management.

Controls To Play Below:

  • Arrow Keys to rotate camera
  • WASD Keys to move rover
  • 'E' to use rover laser
  • 'Q' to reset rover
  • Mouse to use UI and interactable objects

Some things I worked on include using Publisher/Subscriber as the main way for the game objects and UI to communicate with each other - clicking on buildplates or UI buttons triggers events that ensure the buildplates store their respective data and that the UI displays the proper information. I also played around with the particle system and shader graph (to make the terrain).
