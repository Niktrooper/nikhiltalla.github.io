Instagram Clone

The Instagram Clone is a recreation of the popular social media app Instagram. Using clint-side dynamic pages, the website works across multiple users - syncing data in a database and appearing locally on the user seamlessly. It supports all the core instagram features including sign in, find and follow other users, make posts, and even liking and commenting on other posts.

The main skills and tools needed for this project was use of HTML, REST API, React, Java Script, and Python. HTML templates were created and used by React and Java Script to create dynamic pages from templates. REST API was used for handling POST and GET requests to the database storing the users, posts, and other data. The code was all held together by Python code.

This allowed me to learn a lot about working with backend web and databases and how to even make my own website.
